# 自动更新 iconfont 的工具

## 项目根目录下新建 iconfont.json

``` json
{
  "pid": "iconfont 上自己项目的 pid",
  "cookie": "你的 iconfont 中的 EGG_SESS_ICONFONT 这个 cookie 的值",
  "path": "最后文件放入的文件名或者地址"
}
```
